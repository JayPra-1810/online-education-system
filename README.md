# 🎓 Online Education System

A web-based **Online Education System** developed using **PHP**, **MySQL**, **HTML**, **CSS**, and **JavaScript** following the **MVC (Model-View-Controller)** architecture. The system provides an interactive platform for students, instructors, and administrators to manage courses, enrollments, and learning resources efficiently.

## 🚀 Project Overview

The Online Education System is designed to simplify the management of online learning. It allows students to browse and enroll in courses, instructors to manage educational content, and administrators to oversee the entire platform through a centralized dashboard.

## ✨ Features

* Secure User Authentication (Login & Registration)
* Student Dashboard
* Instructor Dashboard
* Admin Dashboard
* Course Management
* Student Enrollment
* User Profile Management
* File Upload Support
* Responsive User Interface
* Role-Based Access Control
* MVC Architecture for Better Code Organization

## ⚙️ Installation

1. Clone the repository.

```bash
git clone https://github.com/JayPra-1810/online-education-system.git
```

2. Move the project into the **htdocs** directory of XAMPP.

3. Start **Apache** and **MySQL** using the XAMPP Control Panel.

4. Create a MySQL database.

5. Import the SQL database file (if available).

6. Update the database configuration in:

```text
config/config.php
```

7. Open the project in your browser:

```text
http://localhost/DivyProject/public
```

## 📌 Note

The SQL database file is **not included** in this repository. Please create and import the required MySQL database before running the application.
